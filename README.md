# Bootlet-Switch

Bootlet-Switch is a lightweight physical toggle switch that works with [Bootlet](https://github.com/mincomk/bootlet). You can select, say, Windows or Linux at boot time by simply flipping the switch.

## Requirements
- STM32 "Blue Pill" development board or equivalent

## Setup
1. Firmware: Refer [firmware/README.md](firmware/README.md) for instructions on flashing the Bootlet-Switch firmware.
2. Bootlet: Follow the [Bootlet setup instructions](bootlet-setup/README.md) to configure Bootlet on your target device.
